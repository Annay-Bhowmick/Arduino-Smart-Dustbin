# Arduino-Based Smart Dustbin 🚮

This project is a smart dustbin that automatically opens its lid when an object is detected nearby using an ultrasonic sensor.

## Components Used
- Arduino Nano
- Ultrasonic Sensor (HC-SR04)
- Servo Motor
- Jumper Wires

## Working Principle
The ultrasonic sensor measures the distance of an object.
If the distance is less than a set threshold (e.g., 20 cm), the servo motor rotates and opens the lid.
After a few seconds, the lid closes automatically.

## Applications
- Touchless waste disposal
- Hygiene-focused environments
- Smart home systems
